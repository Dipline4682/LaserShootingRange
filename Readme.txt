# Кнопки нажимаются двойным кликом
####

####
## Окно настроек
## Окно появляется изначально в режиме H/S/V этот режим в разработке но ползунки можно подвигать если интересно =).
# Ползунки именуются как h1, s1, v1, h2, s2, v2
## Режим окна Lazer. Настраивается обнаружения лазера. Ползунки именуются как Lh1, Ls1, Lv1, Lh2, Ls2, Lv2
# Что бы настроить обнаружение лазера, нужно засветить область мишени и передвигать ползунки до тех пор, пока не будет выдна
# (более менее отчётливо) маленькая точка лазера.
####

####
## Главное окно
## IPS - показывает в нижнем углу скорость обработки информации
## Start - предоставить выбрать область мишени. Сначала выберается верхний левый угол, потом нижний правый. Также глы выбераются двойным кликом
# Встроена защита от дураков если выбрали не правильно углы. просит повторить ввод
# После выбора угла появится кнопка назад, если выделили не ту область.
## Что бы начать играть нужно ещё раз нажать на Start, после того как выбрали область мишени
####

####
# Горячие клавиши. Раскладка клавиатуры должна быть английской
## R - В режиме игры увеличивает окно мишени, что бы, можно было рассмотреть результат. 
# (так же в увеличенном размере можно стрелять и будет всё высвечиваться)
## ESC - Выход из режима игры(стрельбы). Вернуться назад в главное меню
## Q - Загрыть программу
## C - Сброс стрельбы(можно использовать только во время игры)
